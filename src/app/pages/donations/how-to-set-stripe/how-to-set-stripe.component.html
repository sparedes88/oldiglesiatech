Cómo crear tu propia cuenta de Stripe.

Nota:

Stripe colecta el 2.7% de cada transacción + 30 centavos de cada transacción. Puede ser que suene como mucho, pero es mejor tener toda opción disponible a sus donantes. Este dinero va directamente a la organización 'Stripe'. Nuestra plataforma de Donaciones es GRATUITA.
Igualmete creamos un botón para que el donante pueda cubrir ese gasto y su organización pueda tener todo el fondo.
Zelle, Paypal y otras opciones, aunque son gratuitas, pueden limitar al donante al momento de tomar una decisión.
